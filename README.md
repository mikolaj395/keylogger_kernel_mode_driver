# Kernel Mode Keyboard Class Driver
After installing driver logs each key hit into the file, which name is specified in code

## Installation
Before installation you have to enable test signing on your machine and sign driver with certificate generated by Visual Studio. Then the driver hes to be installed. Devcon tool can be used to do so. During installation it is recommended to specify PS/2 keyboard as the device.

## Base
This code is a modification of kbfiltr Keyboard Class Filter Driver published at https://github.com/Microsoft/Windows-driver-samples
